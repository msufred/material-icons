# material-icons
A library of icons based on Material Icons of Google for JavaFx applications.

## Usage:
You can add icons to almost all of the JavaFx controls by using the `setGraphic(Node value)` method.

Examples:
```java
Button myButton = new Button("Click Here");
myButton.setGraphic(new AddIcon(14));

Label lblSearchIcon = new Label("Search);
lblSearchIcon.setGraphic(new SearchIcon(19));
```
